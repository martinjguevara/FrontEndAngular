Portafolio Web

Desarrollo de una aplicación web. Para FrontEnd: HTML y CSS con framework Bootstrap para estructura y estilos y Typescript con framework Angular
para programación. Para BackEnd: Java con framework Springboot para programación y MySQL para base de datos.

Proyecto iluminación Domótica

Circuito que por medio de una aplicación web desde el celular móvil se puede controlar el encendido/apagado de un foco led. Consiste en una
placa Arduino UNO conectada a un módulo WiFi (ESP8266), que a su vez, está conectado a un módulo relé (Circuito MOC3021) y al foco Led.
